How this works

AI providers bill by the token — small chunks of text roughly 0.75 English words each. Every request has two billable parts: the input (your prompt, system instructions, and any context you attach) and the output (the model's reply). Providers publish separate prices for each, quoted in US dollars per million tokens, and output tokens usually cost three to five times more than input tokens.

This calculator multiplies your average input tokens by each model's input price, does the same for output tokens, and adds the two to get a cost per request. Multiplying by your daily request count gives the daily figure, and a 30-day month gives the monthly estimate — the number that matters when you compare models or plan a budget.

Two caveats. First, real traffic is rarely uniform: prompts grow as you add context or chat history, so measure a genuine sample of your workload before committing to a model. Second, prices change frequently, and many providers discount cached prompts or batch processing by 50% or more — so treat the defaults here as a starting point, and paste in current numbers from each provider's pricing page before making a real decision.

Frequently asked questions

How is the cost of an AI API call calculated?

Cost per call = (input tokens ÷ 1,000,000 × input price) + (output tokens ÷ 1,000,000 × output price). For example, 2,000 input tokens and 500 output tokens on a model priced at $3 in / $15 out per million tokens costs $0.006 + $0.0075 = $0.0135 per request.

Are the default prices in this calculator accurate?

They are realistic examples, not live data. Providers change pricing frequently and offer discounts for caching, batching, and volume commitments. Every price cell is editable — always confirm the current numbers on the official Anthropic, OpenAI, and Google pricing pages before deciding.

How many tokens is a typical request?

One token is roughly 0.75 English words, so 1,000 tokens is about 750 words. A short chat prompt might be 100–500 tokens, while a request that includes documents, chat history, or code context can easily reach 5,000–50,000 input tokens. Check your provider's usage dashboard for your real averages.

Why do output tokens cost more than input tokens?

Generating text is more computationally expensive than reading it: the model produces output one token at a time, while input can be processed in parallel. That is why most providers charge 3–5× more per output token, and why limiting response length is one of the easiest ways to cut costs.

Does this calculator send my data anywhere?

No. It is a single HTML file with no analytics, no external requests, and no server. All calculations run locally in your browser, and nothing you type is stored or transmitted.
